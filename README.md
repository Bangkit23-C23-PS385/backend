# Backend System

## Try Locally
1. Clone this repo
2. cd ./backend
3. go mod tidy
4. go run src/main.go

## API Documentation
After running the app, go to this web:
[Swagger Docs](http://localhost:9000/swagger/index.html#/)

Alternative link: http://localhost:9000/swagger/index.html#/
